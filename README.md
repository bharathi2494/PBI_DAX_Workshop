# PBI_DAX_Workshop
## Overview
This workshop will take you from basic DAX fundamentals to advanced time intelligence calculations through progressive hands-on exercises.

## Introduction to DAX & Basic Calculations  
## What is DAX?
- Data Analysis Expressions - formula language for Power BI
- Used to create calculated columns, measures, and calculated tables
- Context-aware: Row Context vs Filter Context

## DAX Syntax Basics:
**Measure Name = FUNCTION(Table[Column])**

## Calculated Columns vs Measures:
- **Calculated Columns**: Evaluated row-by-row during data refresh, stored in model
- **Measures**: Evaluated at query time based on filter context, not stored
